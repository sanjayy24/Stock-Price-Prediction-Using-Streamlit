# 📈 Stock Price Prediction Using Streamlit & Machine Learning

## 📖 Project Overview

This project is a web-based Stock Price Prediction System developed using Machine Learning and Streamlit. It predicts future stock prices by analyzing historical stock market data. The application provides an interactive dashboard where users can select stocks, view historical trends, and obtain predicted prices.

---

## ✨ Features

- 📊 Predict future stock prices using Machine Learning
- 📈 Visualize historical stock price trends
- 🤖 Multiple ML models:
  - Linear Regression
  - Random Forest
  - Support Vector Machine (SVM)
  - Long Short-Term Memory (LSTM)
- 📉 Interactive graphs and charts
- 💻 User-friendly Streamlit web interface
- ⚡ Fast prediction with real-time visualization

## 🚀 How to Run

### 1. Clone the repository

```bash
git clone https://github.com/sanjayy24/Stock-Price-Prediction-Using-Streamlit.git
```

### 2. Navigate to the project folder

```bash
cd Stock-Price-Prediction-Using-Streamlit
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Streamlit application

```bash
streamlit run app.py
```

### 5. Open in your browser

```
http://localhost:8501
```
